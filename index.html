Graduation Internship

For my graduation, I worked for the Smart Education Lab of the AUAS. The project that I worked on is a project for TransEET. My work consisted of making an prototype for an Educational Game that makes use of Augmented Reality. The prototype was made in Unity using AR Foundation (Unity's AR API). This prototype is meant to learn the basics of geometric formulas of surfaces of 3D geometrical objects. The objective is to make 3D geometrical figures dissapear by using the correct formula per surface. This prototype was also presented at two differnt symposiums. The first symposium was the Second International Symposium on Augmented and Virtual Reality in Mathematics Education. The second symposium was the Symposium Smart Education. At both symposium there was interest shown in the prototype.

I have worked on several projects on my own. Some of these can been seen on Github but usually I'm more interested in a certain feature and try to make something with that but I never really develop that into a bigger project. So most of these projects tends to be small.
I do have a bigger project that I worked on for a while. This project came to be alive when Covid struck since it became alot more difficult to get an internship and got a lot more spare time. At the time I wanted a way to save data on shows and movies in JSON format and where the application was offline. This was so that I wouldn't have to worry about servers and I could still access all the data I needed about the shows I had watched or was planning on watching. It started as a purely console application but once it was finished it did include multi-language support and functionality to acces certain websites with the name of the show to get more information online. After this I decided to format it to a Windows Forms Application of which took more time than anticipated.
